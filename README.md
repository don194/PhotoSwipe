# 瞬划

<div align="right"><a href="README.en.md">English</a></div>

## 项目介绍

今年618换手机时，我发现相册里积攒了上万张截图。大部分是向朋友分享信息时残留的无用图片但是也其中有不少是游戏里的精彩瞬间，全部删除又觉得可惜；而系统相册逐张选择删除又很麻烦。

瞬划（PhotoSwipe）因此而来：交互灵感来自交友软件的卡片式左右滑动。快速浏览图片时，左滑标记删除，右滑保留，双指捏合放大方便查看长截图；浏览结束后再统一确认删除，让清理截图更轻松。

## 纯本地处理

瞬划只读取和整理设备本地图片，不联网、不登录、不上传任何图片或使用数据。删除操作会交由 Android 系统显示确认，由你决定是否执行；应用不会在确认前删除图片。

## 下载

前往 [GitHub Releases](../../releases) 下载最新版本的 APK。

## 核心功能

- 主要功能是清理截图，也可按相册整理本地照片
- 左滑标记删除，右滑保留，支持撤销
- 支持拖动和缩放查看图片细节
- 所有图片处理均在设备本地完成
